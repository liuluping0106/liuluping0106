Hugo短命令

2020年七月11日 发布在 博客
正在加载今日诗词...

Hugo短命令实现Bilibili视频自适应屏幕啦。Hugo短命令真的强大丰富且便捷。

技术站点

  • 自适应大小
  • Hugo 短命令
  • video 加字幕
  • srt2vtt

视频效果

代码

<div style="position: relative; padding: 32% 45%;">
    <iframe style="position: absolute; width: 100%; height: 100%; left: 0; top: 0;" 
    src="//player.bilibili.com/player.html?bvid={{.Get 0 }}&page={{ if .Get 1 }}{{.Get 1}}{{ else }}1{{end}}&as_wide=1&high_quality=1&danmaku=1" 
    scrolling="no" frameborder="no" framespacing="0" allowfullscreen="true"></iframe>
</div>

网盘视频

  • <video>标签支持网盘视频和字幕
  • 腾讯云开启CORS
  • 加载video.js插件

效果

代码

<video
   class="video-js vjs-big-play-centered vjs-fluid"
   controls
   preload="auto"
   poster="https://cdn.jsdelivr.net/gh/liuluping0106/sourcebed@master/image/UwOm5D.jpg"
   data-setup="{}" >
   <source src={{.Get 0 }} type="video/mp4" />
   {{ if .Get 1 }}<track src= {{.Get 1}} srclang="By Mydrive"  kind="subtitles" default>
    {{ else }}{{end}}
</video>

PDF效果

代码

<iframe id="pdfiframe" 
    src="https://www.liulp.club/pdf/web/viewer.html?file=https://cdn.jsdelivr.net/gh/liuluping0106/sourcebed@master/document/{{.Get 0 }}.pdf" 
    width="100%" height="100%">
</iframe>

问卷效果

代码

<script type='text/javascript' 
    src='https://www.wjx.cn/handler/jqemed.ashx?activity={{.Get 0 }}&width=700&source=iframe'>
</script>
