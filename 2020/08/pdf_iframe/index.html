网站解锁iframe方案

iframe，简直是扩展的first choice，各种GitHub上的轮子用起来不要太舒服，配合shortcode更佳欧！

pdf.js内嵌网页

<iframe id="pdfiframe" 
    src="https://www.liulp.club/pdf/web/viewer.html?file=https://cdn.jsdelivr.net/gh/liuluping0106/sourcebed@master/document/{{.Get 0 }}.pdf" 
    width="100%" height="100%">
</iframe>

epub.js内嵌网页

<iframe id="epubiframe" allowfullscreen="true" 
    src="https://www.liulp.club/epub/index.html?bookPath=https://cdn.jsdelivr.net/gh/liuluping0106/sourcebed@master/document/{{.Get 0 }}.epub"
    width="100%" height="100%">
</iframe>

MD5密文内嵌

<iframe id="md5iframe"
    src="https://www.liulp.club/md5/decode.html?txt={{.Get 0 }}"
    width="100%" height="300" frameborder="0">
</iframe>

pdfobject方法

此方法无法适配手机浏览器

<object id= "pdfcontainer" data='https://pdfobject.com/pdf/sample-3pp.pdf' 
    type='application/pdf' 
    width='100%' 
    height='600px'>
</object>
