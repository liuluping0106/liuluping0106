2021跨年记(娱乐篇)

2021年一月1日 发布在 博客
正在加载今日诗词...

迎着初升的太阳，2021的大门已经打开。在这个新年与生日相遇的日子里，记录自己2020年的探求与欢愉。

序

想想自己过去的一年，在开心这件事情上，好像没有什么惊天动地的大动作，但细小的点滴却无处不在，时间也变得活泼起来。

书籍

过去的一年，读了一些书。一是春夏学期考试结束，读了一波。然后保研之后，又读了一波。哈哈，总的来说还是，持续性咸鱼，爆发性读书。要数数看了什么书，就有想看很久的《明朝那些事儿》，看了无数遍电视剧第一次看原著的《亮剑》，关于文革始末的《大动乱的年代》，第一次追完的网文《第一序列》以及考古经典网文《凡人修仙传》和《凡人修仙传仙界篇》。

以前，谈起明朝只知道有老不上朝的皇帝。看了《明朝那些事儿》，那历史上的一个个人物渐渐丰满了起来，他们的故事，如今读起来，仍惹人深思。以前，只知道亮剑中抗日救国的热血故事，读了原著才看到其结局那浓密的忧伤，这也激发了我的兴趣去阅读文革相关的书记。《大动乱的年代》也许不能真的让我完全探秘文革风云，但至少为文革梳理了一个发展的脉络，让我们有机会再看当年的混沌与探索。高考结束也考古过好多的网络小说，今年第一次真正追完了一本小说，会说话的肘子的文风，真的非常讨喜，幽默又不乏深思。从《大王饶命》到《第一序列》，能看到肘子仍在不断地学习，探索与突破。真心希望肘子2021能在创作更好的文章。

游戏

想着从最初的红警，到同学们一起玩4399，摩尔庄园和赛尔号。时光荏苒，如今一转眼长大了，在周围同学的吸引下，今年玩起了王者荣耀。农药玩的比较崩，常常被教做人，但也渐渐升了段位，积累的不少英雄不少皮肤，想想还是非常有趣的。

steam上也有了游戏若干。steam上的游戏真的给我开了一扇新世界的大门，原来有这么多好玩的游戏。可惜事实上，我的电脑基本上带不动。不过，感兴趣的很多，有糖豆人，隐形守护者，杀手，PolyBridge2，Overcooked2以及巨烧时间的Stellaris和巨烧硬件的赛博朋克2077。

2021换电脑，务必都玩一遍，哈哈哈哈~~~

影视

今年因为疫情，电影院基本没去过。但不妨碍我看了不少让人印象深刻的影视剧。有治愈系的《凪的新生活》，有恋爱甜剧《世界上最难的恋爱》，有星战系列新作《曼德罗人》第二季，有刚刚开看的《西部世界》，有火锅停不下来的《生活如沸》，有新奇体验的《富豪谷底求翻身》，有《二十不惑》却《三十而已》，也有考古老剧《安家》。也许，影视剧里的人生并不真实，但将自己生活的点与影视剧里的点连成线段，线段的一部分总是落在真实以内，自己的生活以外吧。

想来2020是确实是如同电影的一年，疫情降临，加上澳洲山火，非洲蝗灾，总觉得生活似乎在演绎《流浪地球》里的故事。正好发现了一个用心的年度混剪，正与我当时的想法不谋而合，分享给有缘人。影视和现实可能以我们想不到的方式交会在一起。但希望我们总会是 Happy Ending。

秋冬交替之际，本来打算去看电影，却意外看到了开心麻花的舞台剧。从此一发而不可收拾，从《了不起的爹地》，《乌龙山伯爵》，《莎士比亚别生气》到《贼想得到你》。小半年下来，竟然没有错过开心麻花杭州站的每一场演出。另外，还被yh同学带入坑了音乐剧Les Misérables，真的十分十分的惊艳，让人看了多少遍都还想再看一遍。虽然看过原著，但是音乐剧版本确实是另一种美妙的展开方式。

技术

春夏考试周结束，我一时兴起建立一个小站，从最初只能写简单的博客，到各种更加丰富的UI和功能。有尾部Github式标签，独立站内搜索，相关文章推荐，还有支持epub, pdf, video, m3u8的展示等等。看着小站一点点成长丰富，真的开心！站点运行小200天时间，时不时想到新的想添加的功能，也总是遇到各种各样的问题，一个功能常常看着很简单，一实现起来就是一天甚至几天。现在想来，种种细节，真的蛮有成就感。

这个冬天为了玩江南百景图，又折腾起了安卓模拟器，学了简单的lua脚本设计。哈哈，解放双手，自动刷经验。刷经验事小，真正看到自己的脚本work了，是更有成就感。

-- 模拟滑动并按条件点击
x=-1
i=0
while (x==-1) do
    moveTo(1000+i, 700, 1000+i, 300, {["step"] = 20,["ms"] = 70,["index"] = 1,["stop"] = 1})
    mSleep(800)
    x,y = findImage("shan.png", 0, 0, 1600, 900)
end

tap(x,y)
mSleep(800)
